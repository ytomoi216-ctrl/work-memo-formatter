# AI相談用プロンプトメーカー

雑な業務メモを、ChatGPTなどのAIに相談しやすいプロンプトへ変換するWebツールです。

## 公開ページ

GitHub Pagesで公開しています。

## このツールでできること

業務中に発生したメモや相談内容を入力すると、AIに貼り付けやすい依頼文を作成できます。

現在対応しているプロンプトは以下です。

- 社内共有文を作りたい
- 上長相談文を作りたい
- チャット返信文を作りたい

## 主な機能

- 業務メモの入力
- AI相談用プロンプトの作成
- プロンプトのコピー
- 入力内容と出力内容のクリア

## 使用技術

- HTML
- CSS
- JavaScript

## 使い方

1. 業務メモを入力する
2. 作りたいプロンプトの種類を選ぶ
3. 「プロンプトを作成する」を押す
4. 出力されたプロンプトをコピーする
5. ChatGPTなどのAIに貼り付けて相談する
