# AI相談用プロンプトメーカー

雑な業務メモを、ChatGPTなどのAIに相談しやすいプロンプトへ変換するWebツールです。

## 公開ページ

GitHub Pagesで公開しています。

## このツールでできること

業務中に発生したメモや相談内容に加えて、「AIにやってほしいこと」を入力すると、AIに貼り付けやすい依頼文を作成できます。

現在対応しているプロンプトは以下です。

- 社内共有文を作りたい
- 上長相談文を作りたい
- チャット返信文を作りたい
- メール文を作りたい
- 経緯報告文を作りたい

## 主な機能

- 業務メモの入力
- AIにやってほしいことの入力
- AI相談用プロンプトの作成
- プロンプト種類の選択
- トーンの選択
- 出力形式の選択
- AIへの追加指示の選択
- プロンプトのコピー
- 入力内容と出力内容のクリア

## 使用技術

- HTML
- CSS
- JavaScript

## 使い方

1. 業務メモを入力する
2. AIにやってほしいことを入力する
3. 作りたいプロンプトの種類を選ぶ
4. トーンや出力形式を選ぶ
5. 必要に応じてAIへの追加指示を選ぶ
6. 「プロンプトを作成する」を押す
7. 出力されたプロンプトをコピーする
8. ChatGPTなどのAIに貼り付けて相談する
